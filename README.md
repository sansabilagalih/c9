# Install C9

- **Masuk sebagai root.**

```bash
 sudo -i
```

- **Gunakan screen agar C9 tetap jalan walaupun VPS di close.**

```bash
 screen -S c9
```

- **Clone Repository.**

```bash
 git clone https://github.com/sansabilagalih/c9.git
```

- **Masuk ke folder C9.**

```bash
cd c9
```

- **Berikan Permission.**

```bash 
chmod 777 install.sh
```

- **Jalankan untuk menginstall.**

```bash
./install/sh
```
